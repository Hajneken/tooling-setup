## Other Great Tools 

* `fzf` - fuzzy finder 
* `exa` - colorful replacement for `ls` and `tree`
* `mc` - double pane file browser and commander (lightweight total commander/double commander for terminal) 
* `fd` (`fdfind`) - alternative to `find` command
